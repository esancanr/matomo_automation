# Matomo Automation

Automation of resource provisioning in Matomo Analytics using Python and REST API integration.

## Overview

This project provides an automation solution for programmatically managing resources in Matomo Analytics, including:

- User creation
- Website registration
- Authentication token generation

The main objective is to streamline repetitive administrative tasks in environments where multiple analytics instances must be configured and managed efficiently.

## Features

- Automated administrative operations in Matomo
- Integration with Matomo HTTP API
- User and website management through Python scripts
- Dynamic access token generation
- Lightweight and extensible code for DevOps workflows

## Technology Stack

- Python 3.x
- Requests
- Matomo HTTP API
- RESTful services
